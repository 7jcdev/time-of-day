## Features:

- Dynamic Atmosphere.
- Moon and Moon Phases.
- Night Scattering.
- DeepSpace.
- Dynamic Clouds.
- DateTime.
- Sync System DateTime.
- Simple Sun and Moon Positions.
- Realistic Sun and Moon Positions.
- Easy Setup.
---------------------------------------------

## Install:
- Add time-of-day folder in addons folder.
- Reset Godot editor.
- Active plugin(Project/Project Settings/Plugin).
